Conectar Meta Ads

📖 Cómo obtener las credenciales

  1. Crea una Meta App

    Ve a developers.facebook.com → Crear App → Tipo: Business

  2. Añade Marketing API

    En tu app → Productos → Añadir Marketing API

  3. Obtén permisos

    Necesitas: ads_management, ads_read, pages_read_engagement

  4. Genera tu Access Token

    Graph API Explorer → selecciona tu app → genera token de 60 días

  5. Ad Account ID

    business.facebook.com → Cuentas publicitarias → copia act_XXXXX

🔵

Conectar Google Ads

📖 Cómo obtener las credenciales

  1. Activa la Google Ads API

    Ve a Google Ads API → solicita acceso → obtendrás un Developer Token en tu cuenta MCC

  2. Customer ID

    Tu cuenta de Google Ads → menú superior → número con formato 123-456-7890

  3. Access Token (OAuth2)

    Usa OAuth Playground → scope: https://www.googleapis.com/auth/adwords → genera token

🎵

Conectar TikTok Ads

📖 Cómo obtener las credenciales

  1. Crea una app en TikTok for Business

    Ve a TikTok Marketing API → Crear App → tipo: Web

  2. Obtén el Access Token

    En tu app → Authorization → genera un token de larga duración con permisos de Ads Management

  3. Advertiser ID

    ads.tiktok.com → tu cuenta → el número de 19 dígitos en la URL

🔍

Meta Ad Library — Investigación

📖 Cómo obtener el token

  1. Graph API Explorer

    Ve a developers.facebook.com/tools/explorer

  2. Selecciona tu app y permisos

    Elige tu Meta App → añade el permiso ads_read → haz clic en "Generar Token"

  3. Diferencia con Meta Ads

    Este token es solo para la Biblioteca Pública de Anuncios — no necesita permisos de cuenta publicitaria, solo ads_read

  4. Token de larga duración

    Para evitar expiraciones frecuentes, convierte tu token a uno de 60 días siguiendo la guía de tokens de larga duración de Meta
